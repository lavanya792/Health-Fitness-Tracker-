 Health, Fitness & Nutrition Tracker

 A simple and interactive Python + Streamlit web app for tracking daily meals, calorie intake, nutrition values, and step count. The app allows users to maintain personal profiles, set health goals, and view daily progress with charts and summaries.


 Features
	•	 User Login & Signup
	•	 Personal Profiles (age, weight, goals)
	•	 Meal Logging (breakfast, lunch, dinner)
	•	 Automatic Nutrition Calculation using dataset
	•	 Fuzzy matching for similar food names
	•	 Daily Steps Tracking
	•	 Charts & Summaries for calories and steps
	•	Auto-save logs to CSV


 Tech Stack
	•	Python
	•	Streamlit
	•	Pandas
	•	Matplotlib
	•	FuzzyWuzzy
